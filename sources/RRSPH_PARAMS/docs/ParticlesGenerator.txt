ParticlesGenerator (генератор частиц)

1) порядок вызова: 
- вызывается до генератора параметров и до старта эксперимента
2) генерация структуры директорий:
./ имя эксперимента
    - / Params.json  
    - / dump / [starttimestep].csv
3) генерация параметров:
    mandatory:

    - x_mingeom = границы области
    - x_maxgeom = границы области
    - y_mingeom = границы области
    - y_maxgeom = границы области

    - delta = расстояние между частицами
    - ntotal = общее количество частиц
    - nfluid = количество частиц среды
    - nvirt = количество граничных частиц
    optional:
    - rho0 { 1000 } = базовая плотность частиц
    - nwm_particles_start { ntotal } = индекс первой частицы генератора волн
    - nwm_particles_end { ntotal } = индекс следующей за последней частицы генератора волн
    - boundary_layers_num = количество слоёв границ
    - use_chess_order = использование шахматного порядка граничных частиц
    - depth = глубина
    - x_fluid_min = область среды
    - x_fluid_max = область среды
    - y_fluid_min = область среды
    - y_fluid_max = область среды
    - x_fluid_particles = количество частиц среды по оси Х
    - y_fluid_particles = количество частиц серды по оси Y
    - x_boundary_left = область границ
    - x_boundary_right = область границ
    - x_boundary_center = область границ
    - y_boundary_bottom = область границ
    - y_boundary_top = область границ
    - boundary_delta = расстояние между граничными частицами
    - boundary_separation = расстояние между граничными частицами и частицами среды
